# BDF UM+ Outline

BDF UM+ Outline is a composite font that combines M+ BITMAP FONTS and ucs-fonts.
The license conforms to the M+ BITMAP FONTS license.

- bdfUMplus-outline-regular.ttf
- bdfUMplus-outline-bold.ttf
